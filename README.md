# Training Planner (План тренировок)

**Автор:** Семён Алонов Сергеевич

## Описание

Training Planner - это GUI-приложение для планирования и учета тренировок. Позволяет добавлять тренировки с указанием даты, типа и длительности, просматривать список тренировок в виде таблицы, фильтровать записи и сохранять данные в JSON-файл.

## Основные возможности

- ✍️ Добавление новых тренировок с валидацией данных
- 📅 Поддержка различных типов тренировок (кардио, силовая, йога и др.)
- 🔍 Фильтрация по типу тренировки и диапазону дат
- 📊 Отображение статистики (количество, общая и средняя длительность)
- ✏️ Редактирование и удаление тренировок
- 💾 Сохранение и загрузка данных в формате JSON
- ✔️ Проверка корректности ввода данных

## Требования

- Python 3.6 или выше
- Стандартные библиотеки Python (tkinter, json, os, datetime)
